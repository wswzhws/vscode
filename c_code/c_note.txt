==相等;!=不相等;(!:非);
 判断时成立输出1，不成立输出0；
 优先级高于赋值，低于运算；
 5>3==6>4;表示左右两个判断是否为同样的结果，
 如果是输出1，不是输出0；上式输出为1；
 6>5>4,输出为0（——1>4——0）; 
 连续比较从左到右，等于与不等于优先级低于判断大小关系；

if-else语句使用格式：
 if(条件成立)
 {
     …条件成立执行该内容
 } 
 else if(条件二成立)
 {
     …条件二成立执行该内容
 } 
 else 
 {
     …条件不成立执行该内容
 }
 if语句后不加分号，可以不加{}，但执行内容加分号 
 例：if(条件成立)
        条件成立执行内容; 
    else 
        条件不成立执行内容;
 else总和最近的if相匹配；(无括号情况下，括号优先度最高)
 保持使用{}的习惯； if后()里的值只能是1或0；

switch-case语句使用格式：
 switch(控制表达式) //控制表达式是整型变量 
 {
     case 常量： //常量可以是常数，也可以是表达式
     语句;
     break; //开始从case开始，遇到break停止
     ……(上述循环);
     default：
     语句;
     break;
 }

while和do-while语句使用格式：
 while(条件判断) //当
 {
     …条件成立执行该内容; //循环体要有改变的条件
 }

do
{
    执行内容; //至少执行一次
}
while(条件判断);
